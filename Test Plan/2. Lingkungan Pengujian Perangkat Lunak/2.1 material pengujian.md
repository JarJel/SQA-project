# 2.1 Material Pengujian

Pengujian Midnight Finance didukung oleh infrastruktur dan material berikut:

### 2.1.1 Perangkat Keras (Hardware)

| Komponen | Spesifikasi |
|:---|:---|
| Laptop/PC Developer | Processor Quad-Core, RAM 8GB+, OS Windows 10/11 atau macOS |
| Perangkat Mobile (Browser Test) | Android 10+ atau iOS 15+ untuk pengujian responsivitas frontend React.js |

### 2.1.2 Perangkat Lunak & Teknologi

| Komponen | Detail |
|:---|:---|
| **Backend** | Laravel 11, PHP 8.2, MySQL 8.0 |
| **Frontend** | React.js (Vite), JavaScript ES6+ |
| **API Testing Tool** | Postman atau Thunder Client (VS Code Extension) |
| **Database Client** | TablePlus / DBeaver (untuk verifikasi data langsung di DB) |
| **Version Control** | GitHub — `JarJel/SQA-project` |
| **Web Server** | `php artisan serve` (local: `http://localhost:8000`) |
| **Browser** | Google Chrome (latest) untuk pengujian frontend |

### 2.1.3 Data Uji (Test Data)

| Jenis Data | Detail |
|:---|:---|
| Akun Pengguna | Email terdaftar & terverifikasi, email belum terdaftar, email terdaftar belum verifikasi OTP |
| Portofolio | Minimal 1 akun dengan saldo awal Rp 1.000.000 untuk pengujian transaksi |
| Kategori | Minimal 2 kategori income dan 2 expense per user uji |
| Transaksi | Dataset 10+ transaksi campuran income/expense untuk pengujian filter dan analytics |
