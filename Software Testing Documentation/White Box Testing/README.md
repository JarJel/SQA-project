# White Box Testing

## Deskripsi
White Box Testing adalah metode pengujian yang berfokus pada struktur internal dan logika kode program.

## Tujuan
- Memastikan seluruh kode berjalan dengan benar
- Menghindari bug tersembunyi

## Digunakan pada
- Validasi login
- Proses perhitungan
- Logika kondisi sistem

## Teknik
- Statement Coverage
- Branch Coverage
- Path Coverage
- Loop Testing
- Control Flow Testing

## Kesimpulan
White Box Testing membantu developer memastikan logika program berjalan sesuai desain.
