# Statement Coverage

## Deskripsi
Teknik untuk memastikan setiap baris kode dijalankan minimal satu kali.

## Tujuan
Menghindari kode yang tidak pernah dijalankan.

## Contoh
Jika terdapat 10 baris kode, semua harus dieksekusi saat testing.

## Kesimpulan
Semakin tinggi coverage, semakin kecil kemungkinan bug.
