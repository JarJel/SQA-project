# Desk Checking

**Definisi:**
Pengecekan manual yang dilakukan oleh pengembang (programmer) sendiri terhadap kode program sebelum dijalankan di komputer. Ini adalah tahap awal untuk menemukan kesalahan pengetikan atau logika sederhana.

**Checklist Peninjauan:**
| No | Kriteria Pengecekan | Status |
|----|----------------------|--------|
| 1  | Inisialisasi variabel sudah benar | [ ] |
| 2  | Tidak ada typo pada nama fungsi/variabel | [ ] |
| 3  | Logika if-else sudah mencakup semua kondisi | [ ] |
| 4  | Penggunaan operator aritmatika sudah tepat | [ ] |

**Contoh Kasus:**
Memeriksa apakah ada variabel yang dipanggil sebelum dideklarasikan dalam skrip Python.
