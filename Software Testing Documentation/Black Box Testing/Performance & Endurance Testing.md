## **Performance & Endurance Testing**
### **Deskripsi**
Performance & Endurance Testing adalah pengujian untuk mengevaluasi **stabilitas dan kecepatan** sistem saat dijalankan di bawah beban kerja tertentu, baik dalam jangka waktu pendek maupun lama.
### **Tujuan**
* **Mengukur responsivitas:** Memastikan waktu muat (loading) aplikasi tetap cepat bagi pengguna.
* **Mendeteksi Memory Leak:** Memastikan sistem tidak melambat karena penggunaan RAM yang membengkak seiring berjalannya waktu.
* **Menjamin skalabilitas:** Memastikan sistem tidak "down" saat jumlah pengunjung meningkat secara drastis.

### **Digunakan pada**
* **Web Traffic:** Menguji situs saat terjadi lonjakan trafik (seperti saat promo besar atau flash sale).
* **Aplikasi Real-time:** Memastikan sistem chat atau transaksi keuangan tetap stabil tanpa hambatan.
* **Layanan Server:** Mengevaluasi ketahanan server yang harus menyala 24 jam penuh tanpa henti.

### **Teknik**
* **Load Testing:** Memberikan beban kerja normal hingga maksimal untuk melihat performa sistem.
* **Soak Testing:** Menjalankan aplikasi selama berjam-jam atau hari untuk melihat daya tahan jangka panjang.
* **Stress Testing:** Memberikan beban di luar kapasitas untuk melihat bagaimana sistem pulih dari kerusakan.

### **Kesimpulan**
Pengujian ini memastikan aplikasi memiliki **performa yang tangguh** dan tetap stabil meskipun digunakan secara intensif oleh banyak pengguna.
