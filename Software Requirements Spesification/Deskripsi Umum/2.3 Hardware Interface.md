### **2.3 Antarmuka Perangkat Keras (Hardware Interface)**

Bagian ini mendefinisikan persyaratan perangkat keras yang diperlukan untuk menjalankan aplikasi SAPOPOE, baik di sisi penyedia layanan (*server*) maupun sisi pengguna (*user/client*).

#### **2.3.1 Sisi Server (Server-Side)**
[cite_start]Perangkat keras ini digunakan untuk meng-host basis data **MySQL** dan menjalankan logika aplikasi berbasis **Laravel**[cite: 86]. Spesifikasi minimal yang disarankan adalah:
* **Processor**: Minimal Quad-Core Processor (misalnya Intel Xeon atau AMD EPYC) untuk menangani permintaan *request* dari banyak pengguna secara bersamaan.
* **Memory (RAM)**: Minimal 4 GB (disarankan 8 GB ke atas) guna mendukung proses *caching* dan eksekusi query database yang cepat.
* **Storage (Penyimpanan)**: Minimal 20 GB SSD (Solid State Drive) untuk menjamin kecepatan *input/output* data transaksi dan aset pengguna.
* **Koneksi Jaringan**: Kartu jaringan (NIC) yang mendukung kecepatan minimal 100 Mbps agar akses data melalui HTTP tidak mengalami kendala latensi.

#### **2.3.2 Sisi Pengguna (Client-Side)**
[cite_start]Pengguna mengakses aplikasi melalui *web browser* menggunakan perangkat komputasi pribadi[cite: 86]. Spesifikasi minimal meliputi:
* **Perangkat Pintar**: PC, Laptop, atau Smartphone (Android/iOS).
* **Memory (RAM)**: Minimal 2 GB agar peramban web dapat merender antarmuka aplikasi dengan lancar.
* [cite_start]**Input Device**: Keyboard, Mouse, atau Layar Sentuh (*Touchscreen*) sebagai alat utama untuk melakukan input data pada form seperti **Nominal Pokok** dan **Nama Pemberi Pinjaman**[cite: 1, 3].
* **Output Device**: Monitor atau layar perangkat dengan resolusi minimal 1366x768 piksel untuk menampilkan dashboard keuangan dan grafik target impian secara jelas.

#### **2.3.3 Antarmuka Komunikasi**
* **Protokol Jaringan**: Perangkat keras server dan client harus terhubung melalui jaringan berbasis protokol **TCP/IP**.
* **Media Transmisi**: Dapat menggunakan koneksi kabel (Ethernet) maupun nirkabel (Wi-Fi/Seluler) untuk menjamin ketersediaan akses ke server basis data.
