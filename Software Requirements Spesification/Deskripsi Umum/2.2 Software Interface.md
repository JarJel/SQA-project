2.2 Antarmuka Perangkat Lunak (Software Interface)
Bagian ini merinci produk perangkat lunak yang dibutuhkan sebagai komponen pembangun atau yang berinteraksi langsung dengan aplikasi SAPOPOE.

Sistem Operasi

Deskripsi: Lingkungan tempat aplikasi dijalankan.

Spesifikasi: Windows 10/11 (untuk pengembangan) dan Linux Ubuntu Server (untuk deployment).

Web Server (Apache)

Deskripsi: Perangkat lunak yang melayani permintaan HTTP dari browser pengguna ke aplikasi.

Fungsi: Menjalankan skrip PHP dan mengelola akses ke direktori publik aplikasi.

DBMS (MySQL/MariaDB)

Deskripsi: Sistem manajemen basis data relasional.

Fungsi: Digunakan untuk menyimpan data user, tabel transaksi (t_transaksi), tabel anggaran (t_anggaran), dan tabel relasi (t_relasi). Koneksi dilakukan menggunakan protokol standar MySQL lewat driver PDO PHP.

Bahasa Pemrograman dan Framework (PHP & Laravel)

Deskripsi: Logika utama aplikasi dikembangkan menggunakan bahasa pemrograman PHP versi 8.x.

Fungsi: Framework Laravel digunakan untuk menangani Routing, Middleware keamanan, dan ORM (Eloquent) untuk memudahkan komunikasi dengan database.

Web Browser

Deskripsi: Perangkat lunak di sisi klien untuk menampilkan antarmuka aplikasi.

Spesifikasi: Google Chrome, Mozilla Firefox, atau Microsoft Edge versi terbaru yang mendukung standar HTML5, CSS3, dan JavaScript (untuk visualisasi grafik target impian).

Library Visualisasi (misalnya: Chart.js)

Deskripsi: Library JavaScript pihak ketiga.

Fungsi: Digunakan untuk merender grafik progres pada fitur Target Impian dan statistik pengeluaran bulanan.
