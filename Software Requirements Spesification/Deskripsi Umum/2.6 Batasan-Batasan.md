### **2.6 Batasan-Batasan**

Bagian ini menguraikan batasan-batasan teknis dan operasional yang harus diperhatikan dalam pengembangan serta penggunaan aplikasi SAPOPOE:

* [cite_start]**Ketergantungan Jaringan**: Aplikasi ini berbasis web, sehingga memerlukan koneksi internet atau jaringan lokal (LAN) yang stabil agar pengguna dapat mengakses server aplikasi dan basis data secara *real-time*[cite: 71, 131].
* [cite_start]**Keamanan Data**: Sistem tidak menyediakan enkripsi tingkat tinggi untuk data transaksi di dalam database; keamanan data sangat bergantung pada konfigurasi keamanan pada server MySQL dan protokol akses yang digunakan[cite: 71].
* [cite_start]**Kapasitas Penyimpanan**: Volume data yang dapat disimpan dibatasi oleh kapasitas penyimpanan (*storage*) pada server yang digunakan untuk menampung tabel-tabel basis data[cite: 87, 109].
* [cite_start]**Kompatibilitas Peramban**: Aplikasi dioptimalkan untuk berjalan pada peramban web modern (seperti Chrome, Edge, atau Firefox) dan mungkin tidak berfungsi maksimal pada peramban versi lama yang tidak mendukung standar HTML5 dan CSS3[cite: 71, 147].
* [cite_start]**Validasi Input**: Sistem hanya menerima input data numerik untuk nominal uang dan memiliki batasan jumlah karakter tertentu untuk kolom nama relasi atau deskripsi transaksi guna menjaga performa database[cite: 145, 146].
* [cite_start]**Cakupan Pengguna**: Aplikasi saat ini hanya mendukung penggunaan secara individu (personal) dan belum menyediakan fitur kolaborasi antar-pengguna atau sinkronisasi antar-akun yang berbeda[cite: 4, 182].
* [cite_start]**Bahasa dan Mata Uang**: Antarmuka aplikasi menggunakan Bahasa Indonesia dengan format mata uang Rupiah sebagai standar utama dalam setiap perhitungan nominal keuangan[cite: 92].
