### **2.6 Batasan-Batasan**

Bagian ini menguraikan batasan-batasan teknis dan operasional yang harus diperhatikan dalam pengembangan serta penggunaan aplikasi SAPOPOE:

* **Ketergantungan Jaringan:** Aplikasi ini berbasis web, sehingga memerlukan koneksi internet atau jaringan lokal (LAN) yang stabil agar pengguna dapat mengakses server aplikasi dan basis data secara *real-time*.
* **Keamanan Data:** Sistem tidak menyediakan enkripsi tingkat tinggi untuk data transaksi di dalam database; keamanan data sangat bergantung pada konfigurasi keamanan pada server MySQL dan protokol akses yang digunakan.
* **Kapasitas Penyimpanan:** Volume data yang dapat disimpan dibatasi oleh kapasitas penyimpanan (*storage*) pada server yang digunakan untuk menampung tabel-tabel basis data.
* **Kompatibilitas Peramban:** Aplikasi dioptimalkan untuk berjalan pada peramban web modern (seperti Chrome, Edge, atau Firefox) dan mungkin tidak berfungsi maksimal pada peramban versi lama yang tidak mendukung standar HTML5 dan CSS3.
* **Validasi Input:** Sistem hanya menerima input data numerik untuk nominal uang dan memiliki batasan jumlah karakter tertentu untuk kolom nama relasi atau deskripsi transaksi guna menjaga performa database.
* **Cakupan Pengguna:** Aplikasi saat ini hanya mendukung penggunaan secara individu (personal) dan belum menyediakan fitur kolaborasi antar-pengguna atau sinkronisasi antar-akun yang berbeda.
* **Bahasa dan Mata Uang:** Antarmuka aplikasi menggunakan Bahasa Indonesia dengan format mata uang Rupiah sebagai standar utama dalam setiap perhitungan nominal keuangan.
