# Bab 1 — Pendahuluan
## 1.3. System Requirements

---

> *Kabar baiknya: Sapopoe tidak membutuhkan perangkat canggih. Jika Anda sudah bisa membuka YouTube atau berbelanja online, perangkat Anda hampir pasti siap untuk menjalankan Sapopoe.*

---

## Gambaran Umum

Sapopoe adalah aplikasi berbasis **web** — artinya tidak perlu diunduh atau dipasang seperti aplikasi biasa. Anda cukup membukanya melalui peramban (*browser*) di ponsel atau komputer Anda, dan semuanya langsung berjalan.

Halaman ini menjelaskan spesifikasi minimum yang dibutuhkan agar Sapopoe bekerja dengan lancar di perangkat Anda.

---

## 💻 Kebutuhan Perangkat Keras

Sapopoe dirancang agar ringan dan responsif — dapat menyesuaikan tampilannya secara otomatis baik di layar kecil (ponsel) maupun layar besar (komputer/tablet).

### Penggunaan via Ponsel (*Mobile*)

| Komponen | Spesifikasi Minimum | Rekomendasi |
|---|---|---|
| **Sistem Operasi** | Android 8.0 atau iOS 13 | Android 11+ atau iOS 15+ |
| **RAM** | 2 GB | 3 GB atau lebih |
| **Resolusi Layar** | 360 × 640 piksel | 390 × 844 piksel (setara iPhone 12) |
| **Penyimpanan Tersedia** | 100 MB (untuk cache browser) | 200 MB atau lebih |

### Penggunaan via Komputer atau Laptop (*Desktop*)

| Komponen | Spesifikasi Minimum | Rekomendasi |
|---|---|---|
| **Sistem Operasi** | Windows 10, macOS 10.14, atau Linux (Ubuntu 18.04+) | Windows 11 atau macOS Ventura ke atas |
| **RAM** | 2 GB | 4 GB atau lebih |
| **Resolusi Layar** | 1280 × 720 piksel (HD) | 1920 × 1080 piksel (Full HD) |
| **Penyimpanan Tersedia** | 200 MB (untuk cache browser) | 500 MB atau lebih |

> 💡 **Tips:** Jika Sapopoe terasa lambat dimuat, coba tutup tab-tab peramban lain yang tidak digunakan. Ini biasanya sudah cukup untuk mempercepat tampilan.

---

## 🌐 Kebutuhan Perangkat Lunak

Karena Sapopoe berjalan di dalam peramban, satu-satunya perangkat lunak yang benar-benar Anda butuhkan adalah **peramban web yang mutakhir** (*modern browser*).

### Peramban yang Didukung

| Peramban | Versi Minimum | Tingkat Dukungan |
|---|---|---|
| **Google Chrome** | Versi 90 ke atas | ✅ Sangat Direkomendasikan |
| **Mozilla Firefox** | Versi 88 ke atas | ✅ Didukung Penuh |
| **Microsoft Edge** | Versi 90 ke atas | ✅ Didukung Penuh |
| **Safari** (Apple) | Versi 14 ke atas | ✅ Didukung Penuh |
| **Opera** | Versi 76 ke atas | ⚠️ Didukung, tidak diuji penuh |
| **Internet Explorer** | Semua versi | 🚫 Tidak Didukung |

> ⚠️ **Penting:** **Internet Explorer (IE)** sudah tidak lagi mendapat pembaruan resmi dari Microsoft dan tidak didukung oleh Sapopoe. Jika Anda masih menggunakannya, kami sangat menyarankan beralih ke Microsoft Edge yang tampilannya serupa namun jauh lebih modern.

### Cara Mengecek Versi Peramban Anda

Tidak yakin versi berapa peramban Anda saat ini? Caranya mudah:

- **Chrome / Edge / Opera:** Klik ikon tiga titik (⋮) di pojok kanan atas → pilih **Bantuan** → **Tentang Google Chrome** (atau nama peramban Anda)
- **Firefox:** Klik ikon tiga garis (☰) di pojok kanan atas → pilih **Bantuan** → **Tentang Firefox**
- **Safari:** Dari menu bar di bagian atas layar → klik **Safari** → **Tentang Safari**

> 💡 **Tips:** Aktifkan pembaruan otomatis (*auto-update*) pada peramban Anda. Selain menjaga kompatibilitas dengan Sapopoe, pembaruan rutin juga meningkatkan keamanan perangkat Anda secara keseluruhan.

### Pengaturan Peramban yang Diperlukan

Pastikan pengaturan berikut **aktif** di peramban Anda agar Sapopoe berfungsi dengan baik:

- ☑️ **JavaScript diaktifkan** — Sapopoe menggunakannya untuk menampilkan halaman secara dinamis
- ☑️ ***Cookies* diizinkan** — digunakan untuk mengingat sesi login Anda
- ☑️ **Pop-up tidak diblokir** (untuk domain Sapopoe) — diperlukan untuk beberapa notifikasi penting

> 📌 **Catatan:** Sebagian besar peramban modern mengaktifkan semua pengaturan di atas secara bawaan (*default*). Anda kemungkinan tidak perlu mengubah apa pun.

## 📶 Kebutuhan Konektivitas Internet

Sapopoe adalah aplikasi yang berjalan secara *online*. Koneksi internet bukan sekadar pelengkap — melainkan **bagian inti** dari cara kerja aplikasi ini. Berikut penjelasan mengapa:

### Untuk Apa Internet Dibutuhkan?

**1. Proses Login via OTP**
Setiap kali Anda masuk ke akun, Sapopoe mengirimkan kode verifikasi rahasia (OTP) ke Gmail Anda. Proses pengiriman dan pengecekan kode ini membutuhkan koneksi internet yang aktif. Tanpa internet, kode OTP tidak akan terkirim dan Anda tidak dapat masuk ke akun.

**2. Menyimpan Data Transaksi**
Setiap transaksi yang Anda catat — pengeluaran, pemasukan, transfer antar dompet — langsung disimpan ke server Sapopoe secara *real-time*. Ini memastikan data Anda tidak hilang meskipun Anda berganti perangkat atau membersihkan data peramban.

**3. Memuat Tampilan Aplikasi**
Karena Sapopoe adalah aplikasi web, seluruh tampilan — termasuk grafik, tabel, dan ringkasan keuangan — dimuat dari server setiap kali Anda mengaksesnya. Koneksi yang stabil memastikan semua elemen tampil dengan benar dan cepat.

### Kecepatan Internet yang Disarankan

| Jenis Penggunaan | Kecepatan Minimum | Keterangan |
|---|---|---|
| Browsing dan pencatatan biasa | 1 Mbps | Setara koneksi 3G yang stabil |
| Memuat grafik dan laporan | 3 Mbps | Direkomendasikan untuk pengalaman lancar |
| Penggunaan optimal | 5 Mbps ke atas | Setara Wi-Fi rumahan atau 4G/LTE |

> 📌 **Catatan:** Sapopoe tidak memutar video atau file besar, sehingga tidak membutuhkan koneksi yang sangat cepat. Koneksi data seluler **3G yang stabil** sudah cukup untuk penggunaan sehari-hari.

### Apakah Sapopoe Bisa Digunakan Offline?

Saat ini, Sapopoe **belum mendukung mode offline**. Semua fitur — termasuk melihat saldo, mencatat transaksi, dan membuka laporan — membutuhkan koneksi internet yang aktif.

> 💡 **Tips:** Jika Anda sedang dalam perjalanan dan koneksi tidak stabil, catat transaksi sementara di aplikasi catatan ponsel Anda, lalu masukkan ke Sapopoe sesegera mungkin ketika koneksi kembali tersedia.

## ✅ Daftar Periksa Sebelum Mulai

Gunakan daftar berikut untuk memastikan perangkat Anda sudah siap sebelum mulai menggunakan Sapopoe:

- [ ] Perangkat saya menggunakan Android 8+, iOS 13+, Windows 10, atau macOS 10.14 ke atas
- [ ] Saya menggunakan peramban Chrome, Firefox, Edge, atau Safari versi terbaru
- [ ] JavaScript dan *cookies* diaktifkan di peramban saya
- [ ] Saya memiliki akun Gmail yang aktif dan dapat diakses
- [ ] Perangkat saya terhubung ke internet (Wi-Fi atau data seluler minimal 3G)

Jika semua kotak di atas sudah terpenuhi — **Anda siap memulai!** 🎉
