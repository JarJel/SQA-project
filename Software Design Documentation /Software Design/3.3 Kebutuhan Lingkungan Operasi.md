### 3.3 Kebutuhan Lingkungan Operasi (Operating Environment)

Kebutuhan lingkungan operasi menetapkan batasan infrastruktur dan perangkat lunak pendukung yang diperlukan agar aplikasi **Sapopoe** dapat berfungsi secara optimal, aman, dan stabil. Lingkungan ini terbagi menjadi sisi server (*backend*) dan sisi klien (*frontend*).

#### 3.3.1 Lingkungan Sisi Server (Server-Side Environment)
Server bertugas sebagai pusat pemrosesan logika bisnis, penyimpanan data, dan penyedia layanan API. Spesifikasi minimum yang dibutuhkan adalah:

* **Sistem Operasi:** Keluarga Linux (Ubuntu Server 22.04 LTS sangat direkomendasikan) atau Windows Server dengan dukungan PHP.
* **Web Server:** Apache 2.4 atau Nginx 1.20+.
* **Runtime Environment:** PHP versi 8.2 atau lebih tinggi (dengan ekstensi: BCMath, Ctype, Fileinfo, JSON, Mbstring, OpenSSL, PDO, Tokenizer, XML, dan GD/Imagick untuk pengolahan gambar).
* **Database Management System (DBMS):** MySQL versi 8.0 atau MariaDB 10.6+.
* **Memory (RAM):** Minimal 2GB (direkomendasikan 4GB untuk menangani trafik simultan dan proses *image processing*).
* **Storage:** SSD dengan ruang kosong minimal 10GB (bergantung pada pertumbuhan data transaksi dan penyimpanan avatar pengguna).

#### 3.3.2 Lingkungan Sisi Klien (Client-Side Environment)
Pengguna dapat mengakses Sapopoe melalui berbagai perangkat selama memiliki peramban web yang mendukung standar teknologi modern (ES6+, CSS3, HTML5).

* **Peramban Web (Browser):**
    - Google Chrome (versi 100+)
    - Mozilla Firefox (versi 100+)
    - Safari (versi 15+)
    - Microsoft Edge (versi 100+)
* **Perangkat Keras:**
    - **Desktop/Laptop:** Minimal RAM 4GB dengan resolusi layar 1366x768px.
    - **Smartphone/Tablet:** Minimal RAM 2GB dengan sistem operasi Android 10+ atau iOS 14+.

#### 3.3.3 Lingkungan Pengembangan dan Pengujian (Dev & QA Environment)
Untuk keperluan pengembangan dan pemastian kualitas oleh Tim REMAKO, lingkungan berikut harus tersedia:

* **Version Control:** Git (GitHub sebagai repositori utama).
* **Package Manager:** - **Composer** (untuk dependensi Laravel/PHP).
    - **NPM atau Yarn** (untuk dependensi React/JavaScript).
* **Tools Pengujian:** - **Postman atau Insomnia** (untuk pengujian fungsionalitas REST API).
    - **XAMPP/Laragon** (untuk simulasi lingkungan server lokal).

#### 3.3.4 Konfigurasi Jaringan (Network Configuration)
* **Protokol:** Wajib menggunakan HTTPS (SSL/TLS) pada lingkungan produksi untuk melindungi data finansial sensitif.
* **Bandwidth:** Akses internet stabil dengan kecepatan minimal 1 Mbps untuk memastikan proses *upload* foto WebP dan pengambilan data grafik tidak mengalami *timeout*.
