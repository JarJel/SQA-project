### 2.1 Perspektif Produk

Aplikasi **Sapopoe (Private Wealth Management)** adalah sistem perangkat lunak mandiri (*standalone system*) yang dirancang untuk beroperasi dalam arsitektur *Client-Server* berbasis web. Sistem ini tidak dirancang sebagai komponen dari sistem yang lebih besar, melainkan sebuah platform utuh yang mengintegrasikan manajemen data keuangan pribadi dengan layanan keamanan otomatis.

Perspektif produk dari aplikasi Sapopoe diuraikan melalui poin-poin berikut:

#### 2.1.1 Konteks Sistem
Sapopoe diposisikan sebagai "Buku Kas Digital Cerdas" yang menjembatani kebutuhan pencatatan manual dengan otomatisasi kalkulasi finansial. Sistem beroperasi secara independen dari lembaga perbankan (tidak menggunakan *Open Banking API*), sehingga kontrol data sepenuhnya berada di tangan pengguna. Aliran data utama terjadi antara antarmuka pengguna berbasis React.js dan mesin pengolah logika berbasis Laravel yang terintegrasi dengan basis data MySQL.

#### 2.1.2 Antarmuka Pengguna (User Interfaces)
Antarmuka sistem dirancang dengan prinsip *Mobile-First Responsive Design*. Pengguna berinteraksi melalui *browser* modern menggunakan perangkat *desktop* maupun *smartphone*. Karakteristik visual menggunakan tema "Elite Wealth" dengan dominasi warna gelap (*dark mode*) dan aksen emas untuk memberikan kesan eksklusif dan profesional. Navigasi utama dikelompokkan ke dalam *Sidebar* (untuk *desktop*) dan *Hamburger Menu* (untuk *mobile*).

#### 2.1.3 Antarmuka Perangkat Keras (Hardware Interfaces)
Sistem dapat diakses melalui berbagai perangkat keras dengan spesifikasi minimum sebagai berikut:
* **Sisi Klien (User):** Perangkat dengan prosesor minimal *dual-core* dan RAM 2GB yang memiliki akses jaringan internet.
* **Sisi Server (Hosting):** Infrastruktur berbasis *Cloud* atau *Virtual Private Server* (VPS) dengan kapasitas penyimpanan (*storage*) yang cukup untuk menampung *database* dan aset gambar (avatar) pengguna.

#### 2.1.4 Antarmuka Perangkat Lunak (Software Interfaces)
Interaksi komponen perangkat lunak dalam Sapopoe meliputi:
* **Sistem Operasi:** *Cross-platform* (Linux/Windows/macOS) pada sisi server.
* **Browser:** Mendukung Google Chrome, Mozilla Firefox, Safari, dan Microsoft Edge versi terbaru.
* **Database Management System (DBMS):** MySQL versi 8.0 atau lebih tinggi sebagai media penyimpanan data relasional.
* **Layanan Eksternal:** Integrasi dengan protokol SMTP (Simple Mail Transfer Protocol) melalui layanan Gmail API untuk pengiriman kode verifikasi (OTP) dan notifikasi sistem.

#### 2.1.5 Antarmuka Komunikasi (Communications Interfaces)
Komunikasi antar-komponen sistem dilakukan melalui protokol:
* **HTTP/HTTPS:** Digunakan untuk pertukaran data antara *frontend* (React) dan *backend* (Laravel) melalui *Restful API*.
* **JSON (JavaScript Object Notation):** Format standar pertukaran data yang digunakan untuk seluruh respons API.
* **TLS/SSL:** Enkripsi saluran komunikasi untuk memastikan keamanan data sensitif pengguna selama proses pengiriman data ke *server*.

#### 2.1.6 Batasan Operasional
Produk ini bergantung sepenuhnya pada ketersediaan koneksi internet. Karena bersifat *web-based*, efisiensi sistem dipengaruhi oleh latensi jaringan dan spesifikasi mesin *browser* pengguna dalam me-render komponen UI interaktif.
